What is WebAssembly?

High performance, low-level bytecode for the web

Goals

  • Enables code reusability between native and web
  • Bring second life to legacy code

Format

  • WAT (text format)
  • WASM (compact binary format)

Hello World in WAT


(module
    (func (result i32)
        (i32.const 42)
    )
    (export "helloWorld" (func 0))
)
                    
wat2wasm hello.wat -o hello.wasm

const { readFileSync } = require("fs");

const run = async () => {
    const buffer = readFileSync("./hello.wasm");
    const module =  await WebAssembly.compile(buffer);
    const instance = await WebAssembly.instantiate(module, {});
    console.log(instance.exports.hello()); // 42
};

run().catch(console.error);
